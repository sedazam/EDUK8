# EduK8 Happy Home

EduK8 Happy Home is a parent-focused learning dashboard designed for KS1 and KS2 families. It helps parents organise homework, reading, spelling, maths practice, and home learning routines in one clean interface.

## Features

- Parent dashboard
- Child profile management
- Learning task management
- Progress reporting
- Local persistence
- Protected app routes
- Toast notifications
- Supabase-ready backend structure

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Recharts
- Sonner
- Supabase

## Routes

- `/`
- `/login`
- `/signup`
- `/app/dashboard`
- `/app/children`
- `/app/tasks`
- `/app/reports`
- `/app/settings`

## Local setup

```bash
npm install
npm run dev
```
